Подсказка по использованию

Для того что бы добавить продукт в рассчет необходимо:

Ввести название продукта в окно ввода, в формате "Название продукта, необходимые граммы" и нажать кнопку "Добавить"

После чего ваш продукт появится в списке, при необходимости продукт можно убрать из списка, нажав по нему

Приложение так же позволяет производить перерасчет на размер формы, для этого просто выберите необходимый размер формы
